3. 同时输出下列键，从 history 最近相关轮的「据:」中**原样摘录**合法标识（勿臆造；若无基线则输出空数组）：

- "baseline_extraction_history_ids"：字符串数组，UUID，可空。

- "scope_description"：1～3 句中文，概括数据范围特征（含跟进语境时须写清与上一轮集合的关系）。

规则：

- 回答面向业务用户，简洁清晰，可用短列表。

只输出一个 JSON 对象（不要 markdown 代码围栏），恰好一个字符串键：
- "answer"：完整中文答复；可使用 **Markdown** 排版，小标题请用 `## 标题` 或 `### 标题` 便于阅读。
