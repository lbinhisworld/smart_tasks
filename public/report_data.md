# 报告归档数据（report_data）

本文件由前端通过 `GET /report_data.md` 读取。请将**所有已归档报告**放在下方 `json` 代码块中，形式为 **JSON 数组**；数组中每一项为**一个报告的根对象**。

## 字段约定

- 根对象须包含 **`原始内容`** 字符串字段：存放该次提取对应的**日报/附件原文**（与提取历史中的「原始数据」一致）。
- 其余键与模型解析结果 `parsedJson` 根级一致（如 `分公司名称`、`提取日期`、`production_report` 等）。
- 建议保留 **`id`**（与提取历史记录 `id` 相同），便于历史卡片「查看文件」时按 id 命中本条；未命中时界面会用当前历史记录现场合并展示。
